什么是 OpenClaw？

OpenClaw 是一个开源的 AI 助理平台，让你拥有一个 24/7 在线的私人 AI 助理。与其他 AI 工具不同，OpenClaw 是自主运行的——它不需要你不断输入指令，而是可以主动为你执行任务。

🎯 核心特点

  • 自主执行 - 不只是回答问题，还能主动完成任务
  • 多平台接入 - Telegram、Discord、飞书、钉钉等
  • 技能扩展 - 1715+ 社区技能，一键安装
  • 数据自主 - 完全自托管，数据存储在你自己的服务器
  • 持久记忆 - 记住你的偏好和历史对话

AI Agent vs Chatbot

理解 AI Agent 和传统 Chatbot 的区别，是掌握 OpenClaw 的关键。

特性 传统 Chatbot AI Agent (OpenClaw)
交互方式 被动响应，一问一答 主动执行，可自主决策
任务能力 文本回复 可执行代码、调用 API、操作文件
记忆能力 短期对话上下文 长期记忆，记住用户偏好
扩展性 预设规则 技能插件，无限扩展
数据控制 云端服务，数据外泄 自托管，数据自主

💡 关键理解

OpenClaw 不是"更聪明的聊天机器人"，而是能够自主行动的 AI 助手。你可以对它说"帮我整理本周的邮件"，它会主动访问你的邮箱、分类邮件、生成摘要，而不是仅仅告诉你"我可以帮你整理邮件"。

架构概览

了解 OpenClaw 的整体架构，有助于你更好地使用和扩展它。

用户层
Telegram Discord 飞书 钉钉 Web UI
↓
Gateway（网关层）
消息路由、协议转换、用户管理
↓
Brain（大脑层）
AI 模型、推理决策、任务规划
↓
Skills（技能层）
天气查询 代码执行 文件操作 API 调用 +1715 更多

核心组件说明

🌐 Gateway（网关）

负责与外部平台（Telegram、Discord 等）通信，将用户消息转换为内部格式，并将回复发送回用户。

🧠 Brain（大脑）

核心 AI 层，连接大语言模型（Claude、GPT 等），负责理解用户意图、规划任务、生成回复。

🛠️ Skills（技能）

可插拔的功能模块，每个 Skill 提供特定的能力（如查天气、发邮件）。ClawHub 上有 1715+ 社区 Skill。

💾 Memory（记忆）

存储用户偏好、历史对话、重要信息，让 AI 能够"记住"你。
